# 简介

​	基于gitbook的主题下对其模块化拆分。优势在于，便于寻找样式所在位置，可以自定义特殊标记的span或自定义标签样式。

# 使用

​	将文件`tcs.css`与文件夹`tcs`一并放入Typora主题文件夹即可。

> Typora主题文件夹打开方式：文件(F) --> 偏好设置 --> 外观 --> 打开主题文件夹
